# Service Booking System (Full Stack)

This is a basic full stack project built to learn how frontend, backend and database work together.

## Features

- Booking form
- Admin panel
- View bookings
- Delete booking
- Update booking
- Login system
- Node.js backend
- SQLite database

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- SQLite

## Purpose

This project was built for learning full stack development.
More features will be added in future.
